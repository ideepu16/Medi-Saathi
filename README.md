# 💊 Medi Saathi - Smart Medicine Reminder & AI Health Assistant

Medi Saathi is an intelligent web application designed to help users track their medication schedules effortlessly. It features an automated reminder system, a real-time dashboard, and an integrated AI health assistant powered by Google Gemini.

## ✨ Key Features
* **📅 Real-Time Dashboard:** Tracks 'Taken' and 'Pending' medicines for the current day.
* **📸 Rx Snap (AI Prescription Scanner):** Upload an image of a medicine strip or prescription to automatically extract the medicine name and dosage using OCR.
* **⏰ Asynchronous Alarms:** Browser-based notifications and audio alarms trigger exactly when a dose is due.
* **🤖 AI Health Chatbot:** A 24/7 virtual assistant capable of text and voice input to answer basic health queries (with built-in medical disclaimers).

## 🛠️ Technology Stack
* **Backend:** Python, Flask, SQLite3
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
* **AI Integration:** Google Generative AI (Gemini 1.5/2.0 API) via REST
* **APIs Used:** Web Speech API, Web Audio API, Web Notifications API
